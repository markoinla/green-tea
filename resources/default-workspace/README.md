# Green Tea Workspace

This is the **default workspace** that ships with a fresh Green Tea install — it's the first thing a new user sees, meant to onboard them and demonstrate what the app and its built-in agent can do.

This `README.md` is the workspace description: stable, human-authored context about what this workspace is for. It's read fresh and handed to the AI agent on every prompt. The agent keeps its own running notes in `MEMORY.md`.

## What's in here

- **Welcome to Green Tea** (note) — tour of what the built-in agent can do.
- **How Green Tea Works** (canvas) — a diagram explaining the app.
- **Green Tea Docs Page** (html) — sample HTML artifact.
- **Comparison Table** (csv) — sample data-table artifact.
- **MEMORY** (note) — the agent's persistent scratch notes across conversations.

These exist to showcase Green Tea's core building blocks: notes, canvases, HTML artifacts, and CSV tables, all living side-by-side and editable from Finder or in-app.

## Getting started

- Open **Welcome to Green Tea** for a tour of what the built-in agent can do.
- Look at **Comparison Table.csv** to see how non-note artifacts (tables, canvases, images, PDFs) live alongside your notes.
- Open **How Green Tea Works** to see a canvas diagram in action.
- Open the chat sidebar and ask the agent to draft, research, or reorganize anything in here.

Once a real user starts using this workspace for their own project, this README should be replaced with a description of that project instead of this onboarding content.
